# Personal Finance Tracker

A full-stack Personal Finance Tracker built with:

- Node.js
- Express.js
- SQLite
- Vanilla JavaScript
- Chart.js

## Features

• Income and Expense tracking  
• EMI management system  
• Monthly financial analytics dashboard  
• Expense breakdown charts  
• EMI progress tracking  
• Currency formatting (₹ Indian format)  
• Month filtering for finance data  

## Tech Stack

Backend:
Node.js + Express

Database:
SQLite

Frontend:
HTML, CSS, JavaScript

Charts:
Chart.js

## How to Run

1. Clone repository

git clone https://github.com/Ravwork555/finance-tracker.git

2. Install dependencies

npm install

3. Start server

node server.js

4. Open browser

http://localhost:3000
